角色与权限

系统分为三种角色，不同角色看到的功能和权限不同：

撰稿人

创建稿件、使用AI生成内容、提交审核、根据反馈修改稿件

总监

审核撰稿人提交的稿件、给出修改意见、AI辅助审改、通过或打回

副总裁

终审稿件、最终审批通过、查看全局数据看板

稿件工作流程

每篇稿件经过以下流程，确保质量把控：

创建草稿
→

→

→

审核人可以「通过」或「打回」稿件。打回后撰稿人修改后可重新提交。

撰稿人操作指南

创建新稿件

  1. 点击左侧「稿件管理」，再点击右上角「新建稿件」
  2. 填写标题，选择稿件类型（原型稿/公众号版/媒体版）和业务线
  3. 在「素材/要点」框中输入原始素材、会议纪要或写作要点
  4. 点击「AI 生成」，系统将根据素材自动生成符合滴滴品牌风格的稿件
  5. 在正文编辑区域修改润色，满意后点击「保存」

提交审核

  1. 确认稿件内容无误后，在稿件详情页点击「提交审核」
  2. 稿件状态变为「总监审核中」，等待总监审阅
  3. 如被打回，根据审核意见修改后重新提交

总监操作指南

审核稿件

  1. 点击左侧「审核中心」查看待审稿件列表
  2. 点击稿件进入详情，阅读全文
  3. 可选择三种操作：「通过」转给VP审核、「打回」退回修改、「AI审改」智能修改

AI 智能审改

  1. 在审核面板中输入修改指令（如"第三段太长，压缩到3句"）
  2. 也可以点击麦克风按钮，用语音输入修改意见
  3. 点击「AI修改」，系统根据指令自动修改稿件内容
  4. 查看修改前后的对比（红色为删除，绿色为新增）
  5. 满意后点击「通过」提交给VP审核
语音输入支持自然语言，像跟同事说话一样描述修改需求即可。

VP 操作指南

  1. 在「审核中心」查看已通过总监审核的稿件
  2. 进行终审，选择「通过」完成审批，或「打回」退回修改
  3. 通过后的原型稿可生成公众号版和媒体版两个变形版本

常见问题

忘记密码怎么办？

联系管理员重置密码。

AI生成的内容可以直接用吗？

AI生成的内容基于滴滴品牌风格指南，但建议根据具体场景进行人工润色和调整后再提交审核。

稿件被打回后怎么处理？

查看审核意见，在稿件详情页直接修改内容，修改完成后重新点击「提交审核」。

支持手机访问吗？

目前建议使用电脑浏览器访问，获得最佳操作体验。
